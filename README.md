## Quick start
`
git clone https://github.com/duythien0912/do-an-2.git
`

Backend
```
cd App/backend

yarn && yarn start
```

Frontend
```
cd App/frontend

yarn && yarn start
```

---
### App use : 
* Backend: express
* Frontend: reactjs
* Database: mongodb, local store
